# House Price Prediction using Machine Learning

This project predicts house prices using Linear Regression based on the Ames Housing Dataset.

## Dataset
Ames Housing Dataset (Kaggle)

## Features Used
- GrLivArea
- BedroomAbvGr
- FullBath

## Algorithm
- Linear Regression

## Evaluation Metrics
- MAE: 35,788
- MSE: 2,806,426,667
- R2 Score: 0.63

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab
